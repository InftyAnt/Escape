본 프로젝트는 모바일게임 "좀비고등학교"의 스토리모드 시스템을 본따 만든 게임이다.

플레이어는 시작방의 중앙에서 소환되어 좀비를 제거하며 포탈 방까지 도착하여 계속 새로운 층을 올라가야 한다.



1. 직업군 선택

프로그램을 실행하면 가장 먼저 검은 바탕에 조작법과 직업군 설명이 적힌 화면이 나타난다.

플레이어의 이동은 방향키를 이용하며, 공격은 Q키, 아이템 줍기는 C키, 아이템 사용은 숫자키를 이용한다.

역할군은 아처, 메딕, 체어맨, 파이어맨 4개가 있다.

아처는 최대 7칸의 장거리 관통 공격을 하는 화살을 발사한다. 쿨타임은 0.6초이고, 피해량은 20이다.

메딕은 최대 5칸의 단일 스턴 공격을 하는 주사기를 발사한다. 쿨타임은 1초, 피해량은 25이며, 피격당한 좀비는 1초간 정지한다.

체어맨은 바라보는 방향으로 3 x 2칸에 공격을 한다. 쿨타임은 0.5초, 피해량은 25이다.

파이어맨은 바라보는 방향으로 3 x 3칸에 스턴 공격을 한다. 게이지 형태로 사용하는 동안 게이지가 감소하고 사용하지 않는 동안 게이지가 회복된다. 쿨타임은 0.2초, 피해량은 6이다.

4개의 직업군 중 원하는 직업군을 고르고 싶다면 해당 번호의 키를 입력하면 된다. 아처는 1, 메딕은 2, 체어맨은 3, 파이어맨은 4번 키를 입력한다.



2. 게임 플레이

플레이어는 좀비를 처치해 방을 클리어하며 포탈을 타고 새 맵으로 이동해야 한다.

플레이어가 시작 방이나 포탈 방이 아닌, 일반 방이나 보스 방에 입장하면 그 방의 모든 좀비들을 처치할 때까지 다른 방으로의 이동이 불가하다.

플레이어가 좀비와 닿게 되면 플레이어는 체력이 감소하게 되므로, 최대한 좀비와 닿지 않은 채로 모든 좀비를 처치해야 한다.

좀비의 체력은 100이다.

다른 방으로 이동하고 싶다면 해당 방향의 중앙으로 이동하면 된다. 물론 그 방향에 이어진 방이 있는 경우에만 이동이 가능하다.

다른 맵으로 이동하고 싶다면 포탈 방에 도달한 후 포탈 방의 정중앙으로 이동하면 된다. 다른 맵으로 이동 시 기존 맵의 모든 정보가 지워진다.



3. 아이템

좀비를 처치하면 확률적으로 아이템을 드랍한다.

아이템은 다음과 같다.

붕대(흰색 두루마리) : 현재 체력 25 회복

진통제(주황색 통, 녹십자 모양) : 현재 체력 25 회복

응급 키트(빨간색 상자) : 현재 체력 50 회복

종합비타민(주황색 통) : 60초 간 공격력 50% 상향

프로틴(파란색 통) : 현재 체력과 최대 체력 25 증가

드랍된 아이템을 획득하려면 C키를 누른 후 아이템이 있는 위치로 가면 된다. C키를 미리 눌러둔 후 위치로 이동해도 아이템 획득이 가능하나, 아이템 획득 후 다른 아이템을 추가로 획득하려면 C키를 다시 눌러야 한다.

아이템은 언제나 사용 가능하다. 그러나 체력 회복의 경우 최대 체력을 넘지 못한다. 즉 붕대 10개가 있는 상황에서 현재 체력이 최대 체력과 같더라도 붕대 10개를 모두 사용할 수 있으나 그 효과는 없다.



4. 유저 인터페이스

플레이어의 체력은 왼쪽 최상단에 (현재 체력) / (최대 체력) 형태로 체력바와 함께 나타난다.

오른쪽 최상단에는 미니맵이 나타난다.

각 정사각형은 방들의 위치를 표시한 것으로, 일반 방은 회색, 시작 방은 하늘색, 보스 방은 빨간색, 포탈 방은 자주색으로 나타난다.

또한 현재 플레이어가 있는 방은 정사각형의 가운데에 작은 검은색 정사각형이 그려진다.

일반 방과 보스 방을 클리어하면 미니맵에서 해당 방의 정사각형에 초록색 띠가 둘러진다.

왼쪽 체력바 아래에는 인벤토리가 있다. 아이템 획득 시 인벤토리에 추가되며, 인벤토리가 가득 차면 아이템이 드랍되어 있더라도 획득할 수 없다.

아이템을 사용하려면 해당 아이템의 위치에 대응하는 숫자키를 입력해야 한다. 가장 앞쪽의 아이템을 사용하려면 1번 키를, 두번째 아이템을 사용하려면 2번 키를 누르면 된다.

가장 마지막 아이템은 0번 키를 입력하여 사용한다.

오른쪽 미니맵 아래에는 스킬의 쿨타임이 원형으로 나타나 있다.

처음에는 빨간색으로 되어 있으나 스킬을 사용하면 모두 흰색으로 바뀐 후 빨간색이 반시계방향으로 채워진다.

아처, 메딕, 체어맨은 빨간색이 모두 채워진 후 사용할 수 있다.

파이어맨의 경우 게이지량이 빨간색으로 나타나며, Q키를 꾹 누르면 점점 빨간색이 줄어들고, Q키를 놓으면 다시 차 오른다.



5. 게임 종료

기본적으로 플레이어의 체력이 0이 되면 게임은 종료된다.

그러나 게임 플레이 중 의도적으로 게임을 종료하고 싶다면 esc 키를 입력하면 된다.



6. 주의

게임 창에서 포커스가 오랜 시간 벗어난 후 다시 게임 창으로 포커스를 맞추었을 때 게임이 느려질 수 있다.

그러나 이는 일시적인 현상으로 몇 초 지나면 다시 회복된다.
